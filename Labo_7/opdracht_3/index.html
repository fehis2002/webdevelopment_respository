In de onderstaande afbeelding van het css code die gegenereerd wordt krijgt
het rij met de class "alt" een ander achtergrondkleur. De reden waarom de velden
die onbekend zijn geen rode achtergrond hebben, is omdat de specificiteit van de klasse
alt is groter dan de specificeit van de klasse onbekend. Namelijk alt heeft een
specificiteit van 023 en onbekend heeft een specificiteit van 010. Dit zie je onderaaan
in de laatste afbeeldingen.
