Opdracht 1

De return values van window.confirm("weet u het zeker") is true als je op OK
klikt en false als je op cancel, klikt. De return values van
window.prompt("wat is uw naam", "onbekend") is de ingegeven tekst als je op OK klikt.
In dit voorbeeld zou het onbekend zijn. De return value zal null zijn als je op cancel klikt.
